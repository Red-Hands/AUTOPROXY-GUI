# AUTOPROXY-GUI
Automatic Chain proxy in given time period Using GUI 


#INSTALL


1.  git clone https://github.com/blackhacker3/AUTOPROXY-GUI.git
2.  chmod +x AUTOPROXY-GUI
3.  cd AUTOPROXY-GUI
4.  ./install.sh
5.  To Launch Tool type "autoprox" in terminal ....
